# Sample Dataset Description

This directory contains sample datasets used across the optimization notebooks.

## Files

### user_product_interactions.csv

User interaction data with products including various engagement metrics:

Features:
- X₁: Number of product views
- X₂: Number of add-to-cart actions  
- X₃: Number of PDF downloads
- X₄: Number of purchases

Schema:
- Obs. (n): Observation number
- User ID: Unique identifier for users (format: U###)
- Product ID: Un₹ique identifier for products (format: P###)
- X⋅,1 through X⋅,4: Feature columns

Sample size: 10 observations across 3 users and 5 products
